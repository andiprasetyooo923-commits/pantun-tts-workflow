# Pantun TTS Workflow

Repository ini berisi script Python untuk generate suara **cowok & cewek** dari pantun.  
Workflow GitHub Actions otomatis membuat **MP3** setiap kali dijalankan.  
Hasil MP3 bisa langsung digunakan untuk video, Reels, Shorts, atau project multimedia lainnya.

## Fitur
- Generate suara cowok & cewek dari pantun
- Otomatis via GitHub Actions
- MP3 siap pakai untuk video / Reels
- Bisa dikustom teks pantun kapan saja
